Indiana University | Morning Joe Data Visualization Project | Fall 2017

@LauraHKahn

Introduction

More than 2 billion cups of coffee (cafe arabica) are consumed worldwide each day.

The livelihood of 120 million people depends on the coffee supply chain.

But COFFEE is in TROUBLE!

Coffee rust leads to losses of more than $500 million worldwide.

Coffee rust is the main disease that causes plant leaves to turn yellow.

Coffee futures are the second most popular commodity traded worth over $100 billion.

Does coffee rust affect production and futures prices?

Can visualizations be used to help draw conclusions about potential relationships between these variables?

There is no known research on this topic to date.

Scope

The project focuses on rust, production and futures data from Brazil, Colombia and Papua New Guinea from 1989-2013.

These three countries produce 48% of the world's coffee.

Research Questions?

  • Is there a link between coffee rust and the amount of coffee produced?
  • Is there a link between coffee rust and futures prices?

Variables include 337 observations with the following features:

  • Rainfall
  • Temperature
  • Rust percent
  • Production amount
  • Futures prices

Data visualizations help to accept or not accept the following hypotheses:

  • More rain = more coffee rust
  • Higher temperatures = more coffee rust
  • More coffee rust = less production
  • More coffee rust = lower futures prices
  • More coffee production = lower futures prices

Results

From this histogram and kernel density estimate plot, coffee future data ranges mostly from 100-150 US$.

From the line plots of rust versus production and rust versus futures, these variables are probably correlated.

The line plots were drawn with a logarithmic y-scale due to variation in variable data ranges.

A correlation matrix visualization shows the amount each variable is correlated to another variable.

Positive correlation (red) values means variables change in the same direction.

Negative correlation values (blue) means change in the opposite direction.

Finally, a linear regression plot shows that as rust increases, futures prices decrease.

Visualizations Show

  • More rain does not affect coffee rust
  • Higher temperatures = less coffee rust
  • More coffee rust = more production
  • More coffee rust = lower futures prices
  • More coffee production = higher future prices

    • Conclusion

    • More rust = lower futures prices (Accept original hypothesis)
    • Do not accept other original hypotheses
